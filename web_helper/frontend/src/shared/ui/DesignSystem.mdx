# Design System

This design system outlines the visual language and UI/UX standards for the application, ensuring consistency across all features and widgets. It is built upon Tailwind CSS variables and Shadcn UI principles.

## 1. Color Palette

## 2. Typography

We use standard font sizes and weights. The base font size is 14px (`text-sm`) for the application UI to maximize information density.

**Font Family**: System UI (Inter, San Francisco, Segoe UI, etc.)

### Common Text Styles

| Class | Size | Weight | Usage |
| :--- | :--- | :--- | :--- |
| `text-xs` | 12px | 400/500 | Metadata, captions, badges |
| `text-sm` | 14px | 400 | **Base body text**, table content |
| `text-sm font-medium` | 14px | 500 | Labels, button text |
| `text-base font-semibold` | 16px | 600 | Section headers, card titles |
| `text-lg font-bold` | 18px | 700 | Page titles, major headings |

## 3. Spacing & Layout

We follow a 4px grid system.

| Token | Size | Class | Usage |
| :--- | :--- | :--- | :--- |
| 1 | 4px | `p-1`, `m-1`, `gap-1` | Tight grouping (e.g., icons in button) |
| 2 | 8px | `p-2`, `m-2`, `gap-2` | Standard internal spacing (card padding, list items) |
| 3 | 12px | `p-3`, `m-3`, `gap-3` | Relaxed spacing |
| 4 | 16px | `p-4`, `m-4`, `gap-4` | Container padding, section separation |
| 6 | 24px | `p-6`, `m-6`, `gap-6` | Major layout separation |

### Layout Patterns

- **Page Layout**: `h-full flex flex-col` (Full height, column flex)
- **Cards**: `bg-card border rounded-lg p-4`
- **Flex Row (Center)**: `flex items-center gap-2`
- **Flex Col**: `flex flex-col gap-2`

## 4. Rounding (Border Radius)

| Token | CSS Variable | Class | Usage |
| :--- | :--- | :--- | :--- |
| `sm` | `calc(var(--radius) - 4px)` | `rounded-sm` | Small inputs, badges, checkboxes |
| `md` | `calc(var(--radius) - 2px)` | `rounded-md` | Buttons, standard inputs, cards |
| `lg` | `var(--radius)` | `rounded-lg` | Large containers, modals |
| `full` | `9999px` | `rounded-full` | Avatars, pills, status indicators |

## 5. Shadow & Depth

- **Small**: `shadow-sm` (Cards, buttons)
- **Medium**: `shadow` (Dropdowns, popovers)
- **Large**: `shadow-lg` (Modals, floats)
